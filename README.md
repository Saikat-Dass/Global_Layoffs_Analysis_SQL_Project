<h1>Global Layoffs Analysis using SQL</h1>

<h2>Overview :</h2>

This project analyzes global layoffs across industries, countries, and company stages using a real-world dataset. The goal is to perform end-to-end SQL data cleaning, transformation, and exploratory analysis, similar to how a data analyst would handle messy business data in the industry.

<h2>Tools used to dive into the data analyst job market :</h2>

1. MySQL
2. Data Cleaning
3. Solved Analytical Queries
4. WindowFunctions
5. SQL Joins
6. Aggregations
7. SQL Subqueries

<h2>Conclusion :</h2>

The Global Layoffs Analysis Using SQL project demonstrates a complete end-to-end workflow. Starting from a raw, unstructured dataset, the project showcases strong capabilities in data cleaning, transformation and exploratory analysis, all executed using SQL.Through this analysis, key insights were uncovered about global layoff patterns across industries, countries, funding stages and time periods.
